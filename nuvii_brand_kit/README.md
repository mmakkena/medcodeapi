
# nuvii.ai Brand Kit (Light + Dark)

**Mark:** Neural Wave (Blue→Teal, diagonal ↘)  
**Font:** Inter (Modern clean tech)  
**Spacing:** Tight wordmark

## Files
- `nuvii_ai_logo_horizontal_light.svg` — primary logo (light)
- `nuvii_ai_logo_horizontal_dark.svg` — primary logo (dark)
- `nuvii_ai_icon_light.svg` — icon only (light)
- `nuvii_ai_icon_dark.svg` — icon only (dark)
- `og-card.svg` — 1200×630 Open Graph preview
- `tailwind-nuvii.css` — gradient + button classes

## Colors
- Blue `#1A73E8`
- Teal `#10B981`
- Text (navy slate) `#1F2A36`
- Dark bg `#0b1220`

## Usage
- Use the horizontal logo on website headers and decks.
- Use the icon mark for favicons, social avatars, and app icons.
- For CTA buttons and hero backgrounds, apply the `.bg-nuvii-gradient` class.

> Tip: Keep plenty of white space around the mark (~x-height of the "n").
